# ScreenRuler 屏幕标尺

一款类似 Photoshop 标尺功能的 Windows 桌面工具，帮助设计师和开发者在屏幕上进行精确测量和对齐。

![.NET](https://img.shields.io/badge/.NET-5.0-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ 功能特性

- **屏幕标尺** - 在屏幕顶部和左侧显示半透明标尺，支持像素、厘米、英寸三种单位
- **参考线** - 从标尺拖拽创建水平或垂直参考线，辅助设计对齐
- **多显示器支持** - 每个显示器独立管理标尺和参考线
- **鼠标追踪** - 标尺上显示红色指示线指示当前鼠标位置
- **自定义外观** - 可调整参考线颜色、样式（实线/虚线/点线）
- **参考线持久化** - 关闭软件后参考线位置自动保存，下次启动恢复
- **系统托盘** - 可选在系统托盘显示图标，快速访问功能
- **开机启动** - 可选设置开机自动启动

## 📦 安装

### 系统要求

- Windows 10 或更高版本
- [.NET 5.0 运行时](https://dotnet.microsoft.com/download/dotnet/5.0)

### 下载安装

1. 从 [Releases](../../releases) 页面下载最新版本
2. 解压到任意目录
3. 运行 `ScreenRuler.exe`

## 🚀 使用方法

### 创建参考线

**直接拖拽模式（默认）：**
- 从顶部水平标尺向下拖拽 → 创建水平参考线
- 从左侧垂直标尺向右拖拽 → 创建垂直参考线

**Shift+拖拽模式：**
- 按住 `Shift` 键，然后从标尺拖拽创建参考线
- 可在设置中切换拖拽模式

### 移动参考线

- 按住 `Shift` 键，左键拖动参考线到新位置

### 删除参考线

- **方法一：** 按住 `Shift` 键，双击参考线删除
- **方法二：** 按住 `Shift` 键，将参考线拖拽回标尺区域删除
- **方法三：** 右键点击角落按钮，选择"清除参考线"清除所有参考线

### 显示/隐藏

- 点击左上角的眼睛按钮切换标尺和参考线的可见性
- 隐藏状态下眼睛按钮仍然可见，方便恢复显示

### 设置

右键点击角落按钮或托盘图标，选择"设置"：
- **单位** - 像素、厘米、英寸
- **参考线颜色** - 自定义参考线颜色
- **参考线样式** - 实线、虚线、点线
- **拖拽模式** - 直接拖拽或 Shift+拖拽
- **托盘图标** - 是否在系统托盘显示图标
- **开机启动** - 是否开机自动启动
- **参考线持久化** - 是否保存参考线位置

## ⌨️ 快捷键

| 操作 | 快捷键 |
|------|--------|
| 创建参考线 | 从标尺拖拽（或 Shift+拖拽） |
| 移动参考线 | Shift + 左键拖动 |
| 删除参考线 | Shift + 双击 |
| 拖拽删除参考线 | Shift + 拖拽到标尺区域 |

## 🛠️ 从源码构建

```bash
# 克隆仓库
git clone https://github.com/your-username/ScreenRuler.git
cd ScreenRuler

# 构建项目
dotnet build

# 运行
dotnet run --project ScreenRuler
```
